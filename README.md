 ## React useState & useEffect Practice

# Overview

This project was built to practice React Hooks, specifically useState and useEffect. The goal was to understand state management, component re-rendering, and the execution of side effects in React.

# Features

* State management using useState
* Side effects using useEffect
* Local Storage integration
* Basic component structure
* Event handling

# Challenges Faced

* Understanding how data flows between components
* Understanding when useEffect executes
* Managing dependency arrays correctly
* Debugging state updates and re-renders
* Resolving syntax issues while working with React Hooks

# Learning Outcomes

* Improved understanding of React Hooks
* Better understanding of component lifecycle behavior
* Gained experience debugging React applications
* Learned how state updates trigger re-renders

## Technologies Used

* React
* Vite
* JavaScript
* CSS
